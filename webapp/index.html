Manifest (Descriptor) Instructions

sap.app

Attribute Description
id The namespace of our application component.
type Defines what we want to configure.
i18n Defines the path to the resource bundle file.
title Title of the application in handlebars syntax referenced from the app's resource bundle.
description Short description text of what the application does in handlebars syntax references from the app's resource bundle.
applicationVersion The version of the application to be able to easily update the application alter on.

sap.ui

Attribute Description
technology This value specifies the UI technology.
deviceTypes Tells what devices are supported by the app

sap.ui5

Attribute Description
rootView If your specify this parameter, the component will automatically instantiate the view and use it as the root for this component.
dependencies UI libraries used in the application are declared here.
models Can be used to define models and instantiate them when the app starts. Can also be used to define resource bundles and other configurations.
